# HP G160 GAMING MOUSE

![](screen.jpg)

A simple driver for **HP G160 GAMING MOUSE** implemented using `Python`, `pyusb` and `PyQt5`.

## Feature

- Setting keys by **key code**.
- Setting DPI level.
- Setting breath light speed.
- Setting click speed.
- Setting scroll (roller) enable / disable.
